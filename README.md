<br>

```sh
La base de Datos es del Tipo Relacional, se utilizó lenguage SQL, consta de varias tablas que muestran el funcionamiento interno de una clinica, entre las Tablas podemos encontrar:

- Empleados
- Medicos
- Turnos
- Vacaciones
```
